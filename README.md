# RealtyRank Audit App

Replaces the manual PDF audit workflow: paste audit JSON produced by the
`realtyrank-json` Claude skill, and generate a branded multi-page PDF report.

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

See `CLAUDE.md` for the full project brief (schema, brand palette, build order).
